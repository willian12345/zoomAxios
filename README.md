# 刻度轴
![image](screenshot/demo1.png)

运行环境要求
-----------------------------------------------------------------------------------------------------------
node >= v16.13.1
pnpm >= 6.11.0
vite >= 3.1.0
typescript >= 4.6.4

安装
-----------------------------------------------------------------------------------------------------------
pnpm install 


## demo vue3.0

进入 packages/vue-app

```
pnpm dev
```

## demo react

进入 packages/react-app

```
pnpm dev
```
